### Usage ###

## localhost

```shell
sudo npm link --force
```

## download

```shell
npm i wyf-cli -g
```

## use

```shell
wyf-cli
```

## help

```shell
wyf-cli -h 

wyf-cli --help
```

## version

```shell
wyf-cli -V

wyf-cli --version
```

## create

```shell
wyf-cli create project name
# exp
wyf-cli create vite
```
